Methodology

For this project, I used a qualitative approach by researching how social media platforms like TikTok and Goodreads influence reading trends. I gathered academic sources about BookTok and digital reading culture and analyzed how these platforms promote certain books.

I also created a website using HTML and CSS to present both my research and my own book recommendations. I used VS Code and GitHub to build and organize the site. I also used AI tools like Copilot and ChatGPT to help generate starter code, but I edited and adjusted the content myself.
