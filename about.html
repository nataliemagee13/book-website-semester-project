About This Project

Peachy Pages is a beginner-friendly static website created with only HTML and CSS for a college web development project. This site showcases book recommendations and explores how social media influences reading trends in today's digital world.

Our Mission

We believe that reading connects people, inspires growth, and opens doors to new worlds. Peachy Pages exists to celebrate literature and help readers discover their next favorite book—whether it's through personal recommendations or trending social media discoveries.

About the Creator

This project was built as part of a web development course to demonstrate fundamental HTML and CSS skills. It serves as a foundation for a larger platform dedicated to book lovers and literature enthusiasts.
